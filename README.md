# E-Commerce Website
MERN Ecommerce Website

## Set Up

### Clone repo
```bash
git clone https://github.com/Dev-Elie/E-Commerce.git
```

### Install packages & start services
> Backend

```bash
cd E-Commerce/API
npm install
npm start or npm run dev
```
Navigate back to root.

```
cd ..
```
> Frontend
```bash
cd E-Commerce/UI
npm install
npm start or npm run dev
```
